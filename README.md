## Action to setup [Calcit scripting language](https://calcit-lang.org/)

basic usages:

- `cr` command
- `caps` command
- `bundle_calcit` command, when `bundler` is set to `true`

### Usage

```yml
- uses: calcit-lang/setup-cr@0.0.4
  with:
    version: "0.9.1"
    bundler: false
```
